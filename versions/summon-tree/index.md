# 소환의 나무 버전 매니페스트

> **사용법**: "v1 불러줘", "오리지널 불러줘", "다크 버전 작업해줘" 등
> Claude는 이 파일을 먼저 읽고 해당 버전 파일을 로드합니다.

## 활성 버전 (현재 게임에 적용 중)
- **v1-original**

## 버전 목록

### v1-original
- **코드네임**: 오리지널 / 기본
- **파일**: `v1-original.js`
- **상태**: active (현재 게임에 적용 중)
- **설명**: 최초 소환의 나무. SummonTree 클래스, 6부위 파편(head/body/wings/legs/aura/core), 펫 진화 시스템.
- **핵심 기능**:
  - 정령 소환 (조각 6개 → 커먼45%/레어30%/에픽25%)
  - 펫 진화 (레전드 조각 6개 → 레전드 펫)
  - 소환의 나무 UI + 진화 연출
- **생성일**: 2026-02-20
- **메모**: 기준이 되는 원본

### v2-dark
- **코드네임**: 다크 / 다크버전 / dark
- **파일**: `v2-dark.js`
- **상태**: saved
- **설명**: 현재(2026-02-20) 게임에 적용 중인 소환의 나무 스냅샷. v1-original과 동일 코드.
- **핵심 기능**:
  - 정령 소환 (조각 6개 → 커먼45%/레어30%/에픽25%)
  - 펫 진화 (레전드 조각 6개 → 레전드 펫)
  - 소환의 나무 UI + 진화 연출
- **생성일**: 2026-02-20
- **메모**: 향후 다크 테마 변형 작업용 기반

### v3.0-dark
- **코드네임**: 다크 v3.0 / 황금 미믹
- **파일**:
  - `../combat/v3.0-dark-combat-engine.js`
  - `../combat/v3.0-dark-golden-mimic-belly.js`
  - `../core/v3.0-dark-game-state.js`
- **상태**: saved
- **설명**: 방패병 + GOLDMIMIC 풍선 시스템 + 황금 미믹 뱃속 미니게임 완성 버전
- **핵심 기능**:
  - 방패병: 10마리 스폰마다 등장, HP 10배, ATK 0, 풍선에 글자
  - GOLDMIMC 8글자 수집 (중복 방지, 판 넘어 유지)
  - 양쪽 4+4 투명 풍선 자국 HUD
  - 8글자 완성 → 네온사인 회전 → 거대 미믹(1/3화면) → 영웅 잡아먹기
  - 미믹 뱃속 미니게임: 4분 타이머, 아기 미믹 5~10마리 사냥
  - 뱃속 보상: 5등급 랜덤 (커먼~신화), 골드/장비/힐
  - 연출 시작 즉시 전투 정지 + 풍선 리셋
- **생성일**: 2026-02-20
- **메모**: 테스트용 글자 확률 100% (복원 시 combat-engine.js 961행 100→8)
